视流 AI｜详细操作流与前端界面设计

每一步操作之后出现什么界面、用户下一步做什么、系统后台状态如何变化。

目录安装与授权群内首次使用图片/截图整理结果复核与任务确认日报/周报生成Web 工作台特殊 CV 场景异常与低置信飞书卡片Web 工作台异常恢复

总流程

用户发送图片 → 机器人已收到 → 下载资源 → CV/OCR → 图片类型确认 → 结构化整理 → 结果总览 → 任务确认/日报生成/Web 详情 → 同步成功或异常恢复。

安装与授权

G01 · 企业管理员
飞书应用详情页｜应用介绍 + 权限预览

打开视流 AI 的飞书安装链接或应用市场详情页

出现界面：显示产品说明、可处理图片类型、数据处理范围、需要的飞书权限摘要。
主操作：点击「安装到企业」
下一步：进入 G02 授权确认页
异常：若无管理员权限，显示「联系管理员安装」。
G02 · 企业管理员
飞书授权页｜权限确认界面

查看权限清单

出现界面：显示机器人消息、接收消息事件、读取消息资源、发送消息、卡片交互、多维表格写入等权限。
主操作：点击「同意授权」
下一步：进入 G03 选择使用范围
异常：若拒绝，回到应用详情页并显示「未授权，机器人无法接收图片」。
G03 · 企业管理员
Web 配置向导｜成员与群组范围页

选择机器人可用范围

出现界面：左侧显示组织范围，右侧显示已选择群组；默认仅管理员指定的项目群可用。
主操作：点击「下一步：数据处理模式」
下一步：进入 G04 数据处理模式页
异常：未选择范围时禁用下一步。
G04 · 企业管理员
Web 配置向导｜AI 处理模式页

选择 AI 处理模式

出现界面：提供「本地/私有化优先」「云端增强」「混合模式」三项，显示每项会上传的数据类型。
主操作：选择模式并点击「下一步」
下一步：进入 G05 存储策略页
异常：若企业未开通云端增强，显示升级提示但不阻断私有化模式。
G05 · 企业管理员
Web 配置向导｜数据保存策略页

配置图片和 OCR 文本保存策略

出现界面：可设置原图保存时长、OCR 原文保存时长、结构化任务永久保存/定期清理。
主操作：点击「下一步：同步位置」
下一步：进入 G06 同步位置配置
异常：若选择不保存原图，后续详情页只显示缩略图或处理记录。
G06 · 企业管理员
Web 配置向导｜同步目标页

选择任务写入位置

出现界面：提供飞书任务、多维表格、云文档三个目标；可先只开启多维表格。
主操作：点击「连接多维表格」
下一步：进入 G07 多维表格绑定
异常：若没有编辑权限，提示选择其他表或请求表 owner 授权。
G07 · 企业管理员
Web 配置向导｜字段映射页

绑定多维表格

出现界面：展示字段映射：任务标题、负责人、截止时间、优先级、来源群、原文、状态、置信度。
主操作：点击「保存映射」
下一步：进入 G08 测试机器人
异常：字段类型不匹配时标红并给出推荐字段类型。
G08 · 企业管理员
Web 配置向导 + 飞书群｜测试机器人页

发送测试图片

出现界面：页面提示将一张测试截图发送到选定群；群内机器人同步发欢迎卡片。
主操作：点击「发送测试卡片」
下一步：进入 G09 测试结果
异常：测试群不可达时显示 chat_id 错误。
G09 · 企业管理员
Web 配置向导｜测试结果页

查看测试识别结果

出现界面：显示测试图识别出的 1 条摘要、1 条任务、1 条链接，证明 CV/OCR/LLM 链路可用。
主操作：点击「完成安装」
下一步：进入 G10 完成页
异常：如果测试失败，给出日志：资源下载失败/OCR 失败/LLM 超时。
G10 · 企业管理员
Web 配置向导｜安装完成页

完成安装

出现界面：显示三个后续动作：把机器人加入项目群、配置日报时间、邀请成员使用。
主操作：点击「进入工作台」
下一步：进入 W01 工作台首页
异常：无。

A05 · 群成员
飞书群聊｜个人设置卡

@视流助手 设置

出现界面：显示个人默认动作：结果仅自己可见/群内可见、默认负责人、默认同步位置。

图片/截图整理

B01 · 用户
飞书群聊｜用户消息 + 图片缩略图

在飞书群里发送图片，并 @视流助手 整理

出现界面：群内出现用户图片消息；机器人随后回复处理卡片。
主操作：等待机器人回复
下一步：进入 B02 已收到卡片
异常：如果未 @ 机器人且群配置为只响应 @，机器人不处理。
B02 · 机器人
飞书卡片｜已收到/排队中卡片

接收消息并确认任务

出现界面：显示图片缩略图、来源群、发送人、处理编号、预计处理阶段；按钮：取消、仅自己可见。
主操作：自动进入资源下载
下一步：进入 B03 下载中状态
异常：若消息没有资源 key，显示 X03 无法读取图片。
B03 · 机器人
飞书卡片｜下载中状态

下载消息中的图片资源

出现界面：卡片进度条显示「正在读取图片资源」；不显示结果。
主操作：无，系统自动
下一步：进入 B04 CV 预处理
异常：文件超过限制或下载失败进入 X04。
B04 · 系统
飞书卡片｜识别准备中状态

图像预处理

出现界面：进度更新为「正在校正图片、去噪、切片」；如果是长图显示切片数量。
主操作：无，系统自动
下一步：进入 B05 OCR 中
异常：模糊度过高进入 X05。
B05 · 系统
飞书卡片｜OCR 中状态

执行 OCR 和版面分析

出现界面：进度更新为「正在识别文字与版面」；展示已识别行数。
主操作：无，系统自动
下一步：进入 B06 场景分类结果
异常：无文字但有二维码/图表时进入对应视觉分支。
B06 · 机器人
飞书卡片｜图片类型确认卡

返回场景分类结果

出现界面：显示系统判断：聊天截图/表格截图/白板照片/文档截图/设计稿/报表。给出置信度。
主操作：点击「按此类型整理」
下一步：进入 B07 结构化整理中
异常：用户也可点击「换一种类型」进入 B06a 类型选择。
B06a · 用户
飞书卡片｜图片类型选择卡

点击「换一种类型」

出现界面：列表选择器展示可选类型，并解释每种类型输出结果。
主操作：选择类型并确认
下一步：进入 B07 结构化整理中
异常：无。
B07 · 系统
飞书卡片｜整理中状态

LLM 结构化整理

出现界面：进度更新为「正在提取摘要、任务、链接、时间、负责人」；卡片保持可取消。
主操作：无，系统自动
下一步：进入 B08 结果总览卡
异常：模型超时进入 X06，可重试或只返回 OCR 结果。
B08 · 机器人
飞书卡片｜结果总览卡

展示识别结果总览

出现界面：顶部显示摘要；中部显示任务数量、链接数量、风险数量；底部按钮：保存任务、生成日报、打开详情、忽略。
主操作：点击「打开详情」或「保存任务」
下一步：进入 C01 或 W02
异常：若置信度低于阈值，自动进入 X01 低置信卡。
B09 · 用户
飞书卡片｜确认忽略卡

点击「忽略」

出现界面：提示「是否删除本次识别结果？」按钮：删除、仅关闭卡片。
主操作：点击「删除」
下一步：结束流程
异常：无。

飞书卡片规格

FC-01

入群欢迎卡

告诉用户机器人能做什么，并给出第一步入口。

  • 标题：视流助手已加入本群
  • 说明：发送截图/图片/白板/表格并 @我即可整理
  • 示例按钮：整理聊天截图、识别表格、整理白板
  • 设置按钮：个人设置、管理员配置
下一步：点击示例进入 A02；直接发图进入 B01。
FC-02

已收到/处理中卡

让用户知道机器人已经接收任务，避免黑箱等待。

  • 图片缩略图
  • 处理编号
  • 进度条
  • 当前阶段：读取资源/图像预处理/OCR/AI 整理
  • 按钮：取消、仅自己可见
下一步：系统自动更新到图片类型确认卡或结果总览卡。
FC-03

图片类型确认卡

在 CV 场景分类后让用户确认处理路径。

  • 图片类型：聊天截图/表格截图/白板照片/文档截图
  • 置信度
  • 为什么这样判断
  • 按钮：按此类型整理、换一种类型
下一步：确认后进入结构化整理；更换类型后重新走对应 pipeline。
FC-04

结果总览卡

在群里用最少信息展示识别结果，并提供下一步动作。

  • 摘要 3 条以内
  • 任务数、链接数、风险数
  • 高优先级任务预览
  • 按钮：保存任务、生成日报、打开详情、忽略
下一步：保存任务进入任务确认；生成日报进入日报范围；打开详情进入 Web。
FC-05

任务确认卡

把 AI 候选任务变成用户确认的正式任务。

  • 任务列表
  • 负责人
  • 截止时间
  • 优先级
  • 来源原文
  • 置信度
  • 按钮：全部保存、逐条编辑、忽略低置信
下一步：确认后同步到任务系统或多维表格。
FC-06

日报草稿卡

展示日报初稿，并让用户决定编辑或发布。

  • 今日完成
  • 进行中
  • 问题风险
  • 明日计划
  • 按钮：编辑、发到群、生成云文档、复制文本
下一步：编辑进入 Web 报告编辑器；发布后返回成功卡。
FC-07

同步成功卡

明确结果已经写入哪里，用户下一步能做什么。

  • 成功状态
  • 写入目标链接
  • 已保存任务列表
  • 按钮：查看任务、设置提醒、生成日报
下一步：进入任务看板、提醒设置或日报生成。
FC-08

异常处理卡

给出可操作的错误恢复，而不是只报错。

  • 错误原因
  • 影响范围
  • 推荐修复动作
  • 按钮：重试、重新上传、打开权限诊断、取消
下一步：按错误类型回到对应步骤。

Web 工作台页面

页面布局核心 UI下一步
WEB-01 工作台首页顶部指标 + 待处理列表 + 最近识别今日处理量、待确认任务、日报草稿、失败任务点击记录进入识别详情。
WEB-02 识别详情页左图右结果图片预览、OCR 框选、摘要、任务、链接、风险、证据高亮确认任务、编辑任务、加入日报。
WEB-03 任务看板Kanban 四列待确认、进行中、已完成、已忽略；筛选器；同步状态拖拽更新状态或打开任务详情。
WEB-04 链接库卡片列表 + 详情抽屉链接标题、摘要、来源图、标签、加入日报按钮加入日报或打开链接。
WEB-05 日报编辑器左素材右编辑器素材面板、Markdown 编辑区、发布设置保存草稿或发布到群/云文档。
WEB-06 设置与权限诊断分组设置页机器人权限、AI 模式、数据保存、同步字段映射、隐私中心修复权限、调整策略或删除数据。

异常恢复原则

所有异常界面都必须说明三件事：发生了什么、影响什么、用户下一步怎么恢复。不能只显示技术错误码。
